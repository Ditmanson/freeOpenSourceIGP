IGP RULES

ruleset

Basic Requirements for dog behavior

  • Confidence
  • Free, happy, motivated to work
  • Attentive to the handler
  • Harmony of the team whatever that means
  • Correctness
  • Acceptance of commands
  • Basic assesment of execution of trial exercises
  • Response to verbal

Verbal corrections

  • 1 additional command is 1.5 deduction
  • 2nd additional commands are each 2.5 point ———-
  • 3rd additional command is a dq for the round

Praise

May be praised after each exercise, must be back in position within 5 seconds

equipment

  • must use single row large link type collars
  • No muzzles
  • always have a leash
  • No training vests or jackets not sure what about hoodies?

Socrebook

We need a score book from a registered club

Recall

We can use the dogs name or a single command not both. that’s lame..
